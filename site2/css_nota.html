div = Quando usar div?
Resumo. O elemento de divisão HTML < div> é um container genérico para conteúdo de fluxo, que de certa forma
não representa nada. Ele pode ser utilizado para agrupar elementos para fins de estilos (usando class ou id), ou porque
eles compartilham valores de atributos, como lang.

< /div>
